## Site Structure

- index.html — main resume content

- netlify.toml — Netlify deployment config
- vercel.json — Vercel routing config
- README.md — project overview and resume summary

## How to Customize

Edit the content directly in index.html for:
- hero metrics and positioning
- experience timeline entries
- featured work cards
- toolkit sections
- certifications and education

Styling changes can be made in styles.css, including:
- color palette and typography
- layout spacing and responsive behavior
- print-friendly PDF export rules

## Deployment

This site is designed for static hosting and can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting provider

## Notes

The design is intentionally simple and fast-loading, with semantic HTML, accessibility support, and a print-optimized stylesheet for exporting a clean PDF version of the resume.
